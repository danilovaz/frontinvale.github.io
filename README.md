FrontInVale 2016


### Set up & Install dependencies

```
$ git clone git@github.com:frontinvale/frontinvale.github.io.git
$ cd froninvale.github.io
$ gem install jekyll
```

### Server

```
$ jekyll serve
```

### Using a browser, go to:

```
http://localhost:4000
```
