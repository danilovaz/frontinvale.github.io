18:00 - Grid Layout e Flexbox: esqueça o float e seja feliz

Diego Eis

Por muito tempo table e depois float foram as maneiras disponíveis para implementar a estrutura de websites. Mas tables não têm boa performance e dificultam a construção de layouts de forma semântica. Surgiu então a propriedade float, trazendo maior flexibilidade para criação de layouts complexos, melhorando o código HTML e facilitando a manutenção. Porém, float também traz problemas como a dependência do código HTML.
O Grid Layout e o Flexbox são as novas recomendações da W3C para a estruturação de layouts em diversos dispositivos, e solucionam boa parte dos problemas presentes na combinação de table e float. Nessa palestra você verá como o uso de Grid Layout e Flexbox pode agilizar o processo da estruturação de layouts e como revolucionaram a maneira com que montamos a estrutura HTML juntamente com o CSS.
